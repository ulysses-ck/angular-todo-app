## ‚öô Installation

1. Clone repository

```bash
git clone https://github.com/ulysses-ck/dictapp-react
cd angular-todo-app
```

2. Install dependencies

```sh
pnpm install
```

3. Init app

```bash
ng serve --open
```

4. Build for production

```bash
ng build
```
